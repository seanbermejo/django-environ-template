# Django-Environ template

This template creates a simple template structure with django-environ. 

## Structure

```
- config/
 - settings/
   - __init__.py
   - base.py
   - platforms/
     - local.py
 - urls.py
 - wsgi.py
- manage.py
```

## Installation

```
$ django-admin startproject project --template django-environ-structure
```
